About Me

I'm a web developer based in Austin, TX. I initially pursued an education in web development as a way to cultivate an entirely new set of skills, but I ended up falling in love with the way I could use those skills to build incredible things and solve complex problems. I work well with Javascript and have used technologies like Node.js, Express.js, and MongoDB to build well-rounded full-stack applications.


Originally from New Jersey, I've been an honorary Texan for eleven years and a resident of the great city of Austin for five. When I'm not actively coding, I enjoy spending time with my friends, taking advantage of the Austin nightlife, and hiking through the Barton Creek Greenbelt. My greatest aspiration is to travel the world and explore new places. During my undergraduate studies at the University of Texas, I threw myself headfirst into a fully immersive study abroad program in Santander, Spain, so I know how rewarding it can be to persevere after putting myself in challenging, unfamiliar situations.

Recent Work

Giphy API

Application that utilizes the giphy.com API to display 10 gifs after submitting a search request.

Giphy API Generator

This gif generator queries the Giphy.com API for 10 random gifs for each button click. Each gif renders as a still image, but can be animated by clicking the image. The search bar to add more search buttons doesn’t currently function, so that is a feature I’ll be adding in the future.

Technologies I utilized: AJAX, jQuery, JavaScript, Materialize, Responsive Design.

Check it out here: https://giphyapigenerator.herokuapp.com

justGo

justGo

This was a really cool project that was developed between myself and four other team members. This application uses Home Away’s API to take in a user’s search criteria and then render a list of vacation rental listings that matches their input. Current Projects

Katie Bodden Photography

Currently developing a photography site to showcase the portfolio of an Austin-based freelance photographer.

Katie Bodden Photography

This is a personal freelance project that I also used as my full-stack final project submission for The Coding Boot Camp at UT Austin. It’s a simple photography portfolio site developed for an Austin-based photographer that runs on a Node/Express web server. The blog page scrapes posts from a Tumblr blog, saves the content to a MongoDB database, and then renders the content to the HTML page. Most of the images and content are dummy fillers that are intended to be placeholders for the time being. I’m currently working with the client to further develop this site. The gallery images are actual images owned by the client, while the rest are stock images.


Technologies I utilized: Node.js, Express.js, RequireJS, NPM, jQuery, AJAX, MongoDB, Materialize.


Check it out here: https://kbphotography.herokuapp.com

Train Scheduler
