Recent Work

Giphy API

Application that utilizes the giphy.com API to display 10 gifs after submitting a search request.

Giphy API Generator

This gif generator queries the Giphy.com API for 10 random gifs for each button click. Each gif renders as a still image, but can be animated by clicking the image. The search bar to add more search buttons doesn’t currently function, so that is a feature I’ll be adding in the future.

Technologies I utilized: AJAX, jQuery, JavaScript, Materialize, Responsive Design.

Check it out here: https://giphyapigenerator.herokuapp.com

justGo

A collaborative project between myself and four other team members.

justGo

This was a really cool project that was developed between myself and four other team members. This application uses Home Away’s API to take in a user’s search criteria and then render a list of vacation rental listings that matches their input. This allows users to search for home rentals near the types of activities that they plan on doing during their trip. My role in this project was on the front-end team.

Technologies I utilized: JavaScript, Materialize, Responsive Design, Git collaboration.

Check it out here: just-go.herokuapp.com

Train Scheduler

Application that uses Moment.js to calculate trains times and a Firebase database to save train information.

Train Scheduler

This is a train-scheduling app that I started at the beginning of my education at The Coding Boot Camp at UT Austin. As of this writing, it is currently unfinished, but I intend to return to it for future development. It’s intended use is to utilize Moment.js to provide a train schedule that allows the user to view train times and add their own entries if they wish.

Technologies I intend to utilize: jQuery, Moment.js, Firebase, Bootstrap.

Check out my code here: https://github.com/mcheroff/Train-Scheduler

Current Projects

Katie Bodden Photography

Currently developing a photography site to showcase the portfolio of an Austin-based freelance photographer. A work in progress!

Katie Bodden Photography

This is a personal freelance project that I also used as my full-stack final project submission for The Coding Boot Camp at UT Austin. It’s a simple photography portfolio site developed for an Austin-based photographer that runs on a Node/Express web server. The blog page scrapes posts from a Tumblr blog, saves the content to a MongoDB database, and then renders the content to the HTML page. Most of the images and content are dummy fillers that are intended to be placeholders for the time being. I’m currently working with the client to further develop this site. The gallery images are actual images owned by the client, while the rest are stock images.


Technologies I utilized: Node.js, Express.js, RequireJS, NPM, jQuery, AJAX, MongoDB, Materialize.


Check it out here: https://kbphotography.herokuapp.com
